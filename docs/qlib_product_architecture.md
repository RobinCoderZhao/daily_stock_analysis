# Qlib 产品与架构文档

> 微软开源 AI 量化投资平台深度分析

## 1. 项目概述

**Qlib** 是微软研究院开源的、面向 AI 的量化投资平台，旨在利用 AI 技术在量化投资的全链条（从 Alpha 挖掘到订单执行）释放价值。

| 维度 | 说明 |
|------|------|
| **开源协议** | MIT License |
| **开发语言** | Python 3.8 - 3.12 |
| **核心定位** | AI 驱动的全链路量化投资研究平台 |
| **市场支持** | A 股、美股、巴西 Ibovespa |
| **GitHub Stars** | 18k+（极高社区活跃度） |
| **论文支持** | 20+ 篇顶会论文实现 |

## 2. 核心能力

### 2.1 高性能数据引擎

Qlib 的数据引擎是其核心竞争力之一：

- **自研二进制存储格式**：数据以紧凑的 `.bin` 文件存储，无需经过多层数据库接口
- **ExpressionCache + DatasetCache**：多级缓存机制，大幅提升重复查询性能
- **秒级数据加载**：14 个因子 × 800 支股票 × 13 年数据仅需 **7.4 秒**（含缓存），远超 MySQL（365 秒）和 MongoDB（254 秒）
- **支持分钟级高频数据**：1min K 线数据获取和处理
- **在线/离线双模式**：支持本地部署和共享数据服务器

```
性能对比：
HDF5:    184.4s
MySQL:   365.3s
MongoDB: 253.6s
InfluxDB:368.2s
Qlib:      7.4s  ← 50x 提升
```

### 2.3 Alpha 因子体系

内置两套标准因子集：

| 因子集 | 因子数量 | 覆盖 | 说明 |
|--------|---------|------|------|
| **Alpha158** | 158 个 | 美股 + A 股 | 包含 MACD、RSI、KDJ 等技术因子 |
| **Alpha360** | 360 个 | 美股 + A 股 | 扩展因子集，覆盖更多维度 |

因子计算通过 **表达式引擎（Expression Engine）** 实现，支持自定义算子：
```python
# 因子表达式示例
fields = ['$close', '$volume', 'Ref($close, 1)', 'Mean($close, 3)', '$high-$low']
```

### 2.4 策略与回测框架

- **TopK 策略**：基于模型预测信号选择 TopK 股票
- **信号策略**：基于买卖信号的交易策略
- **嵌套决策框架**：支持多层策略嵌套优化（如：日频组合管理 + 分钟级订单执行）
- **完整回测引擎**：含交易成本、滑点、日频/分钟级回测
- **风险分析报告**：IC、ICIR、最大回撤、年化收益、信息比率等指标

### 2.5 强化学习框架

- **订单执行 RL**：支持 TWAP、PPO、OPDS 等智能订单执行策略
- **环境抽象**：提供 `ExecutionEnv` 环境接口
- **多层嵌套**：支持组合管理 + 订单执行的联合优化

### 2.6 在线服务能力

- **模型自动滚动训练**：支持定期重训模型适应市场变化
- **增量数据更新**：支持日频数据自动增量更新
- **在线推理**：低延迟在线模型服务

### 3.1 核心模块说明

| 模块 | 路径 | 职责 |
|------|------|------|
| `qlib/data/` | 数据层 | 二进制存储、缓存、表达式引擎、数据集构建 |
| `qlib/model/` | 模型层 | 模型基类、Trainer、集成方法、风险模型 |
| `qlib/strategy/` | 策略层 | TopK/信号策略、嵌套决策 |
| `qlib/backtest/` | 回测层 | 回测引擎、交易执行器 |
| `qlib/rl/` | 强化学习 | RL 环境、策略、训练 |
| `qlib/workflow/` | 工作流 | 实验管理、记录、可视化 |
| `qlib/contrib/` | 贡献模块 | Alpha 因子、处理器、评估指标 |

### 3.2 工作流机制

Qlib 通过 YAML 配置驱动全流程：

```yaml
# workflow_config.yaml 示例结构
qlib_init:
  provider_uri: "~/.qlib/qlib_data/cn_data"
  region: cn
market: &market csi300
dataset:
  class: DatasetH
  kwargs:
    handler:
      class: Alpha158
model:
  class: LGBModel
strategy:
  class: TopkDropoutStrategy
backtest:
  ...
```

一条命令即可运行完整流程：
```bash
qrun workflow_config_lightgbm_Alpha158.yaml
```

## 7. 局限性

| 局限 | 说明 |
|------|------|
| 数据源受限 | 官方数据集暂不可用，依赖社区或 Yahoo Finance |
| 中文支持 | 文档主要为英文，部分 API 对中文不友好 |
| 实时性 | 设计偏研究导向，实时交易场景需额外开发 |
| 学习曲线 | 概念多、配置复杂，上手有门槛 |
| A 股适配 | 部分模型参数默认针对美股，A 股需调参 |
